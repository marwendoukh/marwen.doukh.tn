Many people asks : "How can I help Mozilla ? I'm not familiar with programming !" . Then answer is "Use Firefox Nightly !".

In this blog post , we'll find out why Nightly is important and what you can do for Mozilla .

What is Nightly ?

In software development process , any program version have a life cycle , like we Humans , Alpha , Beta and release .

Alpha is the version that contains the latest features and technology , it not available to public because it's unstable and under testing.

Beta is almost stable version of the software , but it needs more testing to make sure it does not contain bugs or security issues.

Release is the most stable version of the software and it is available to public . It has been tested for many weeks and almost all the issues were fixed by developers.

Back to the main topic , Nightly is the alpha version of Firefox , it contains the most newest Firefox features that are not available to public ,it is under continuous testing by the community.

How can I help Mozilla ?

As we mentioned above , Nightly is unstable and contains the latest features , so Mozilla should test these features before making it publicly available .When installing Nightly , it will send anonymous data about how much RAM it consumes , how much it takes to start , how many times it crashes and the crash's cause ... All the Mozilla community is working hard every day to make Firefox better whether by coding or testing. So you can be part of this and help to shape the next version of Firefox by simply installing Firefox Nightly and reporting any issue or bug you find.

If you want to test a more stable version of Firefox , you can install Firefox Beta from here , it's the last chance to catch bugs before landing user's computers.

What is coming next ?

I've been using Firefox Nightly for more than a year , and during all this period , I'm receiving a new update every day , YES ! every day ! and I can notice that Firefox is improved day after day , new features are added , speed is improved , new privacy features are included regularly .

Since Mozilla is open , it shares periodically blog posts to tell the community what is coming next and what are the features that were added in the previous weeks , you can follow it by visiting the Firefox Nightly blog.
